## 7. 機能の関係性

### 7.1 データフロー
1. ログイン → ホーム画面
2. シフト管理 → シフト申請
3. シフト申請 → 承認フロー
4. シフト承認 → ガントチャート更新

### 7.2 権限フロー
1. マスター：全機能利用可能
2. 講師：シフト管理・申請機能のみ
3. 申請フロー：講師→マスター→承認
