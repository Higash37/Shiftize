## 3. ガントチャート表示

### 3.1 今月シフト表示
1. シフトの重なり表示
   - 部分的に重複しないシフトは1行に詰める
   - 例：[15:00~17:00][17:00~19:00]

2. 部分的な重複チェック
   - 例：[17:00~18:00]
         [17:30~19:00]

### 3.2 来月シフト表示
1. シフトの1行表示
   - 同じ日でも全てのシフトを1行ずつ表示
   - 例：[15:00~17:00]
           [17:00~19:00]

2. 編集機能
   - ドラッグ＆ドロップ
   - シフトの追加
   - シフトの削除
